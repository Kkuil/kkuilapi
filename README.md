### 1. 拉下代码

```bash
git clone https://gitee.com/Kkuil/kkuilapi.git
```

### 2. 安装环境

#### 2.1 前端

> 本项目使用node16.20.0开发，建议使用16.20.0及以上版本进行开发

```bash
# 进入kkuilapi-frontend目录
cd kkuilapi-frontend
# 安装依赖
npm install
# 启动项目
npm run start
```
